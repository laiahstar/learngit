1.git init  --创建一个新目录，进入该级目录后，把目录变成Git可以管理的仓库
2.git add readme.txt  --编写readme.txt,使用git add添加到Git仓库目录下
3.git commit -m "说明的内容"  --提交文件,-m后面输入的是本次提交的说明
4.git status 要随时掌握工作区的状态
5.如果git status告诉你有文件被修改过，用git diff可以查看修改内容。
